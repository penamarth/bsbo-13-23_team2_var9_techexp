Операция: ArchiveApplication(applicant: Applicant, data: ApplicationData) : Application

Ссылки: Прецеденты: Подача заявки соискателем

Предусловия:
- Существует экземпляр applicant класса Applicant
- Существует экземпляр data класса ApplicationData  
- Сервис GrantSystemService инициализирован и готов к работе
- Инициализирован процесс подачи заявки соискателем

Постусловия:

- Создан экземпляр app класса Application (создание экземпляра)
- Экземпляр app связан с applicant через атрибут ApplicantId (формирование ассоциации)
- Атрибуту app.Data присвоено значение data (модификация атрибута)
- Атрибуту app.Status присвоено значение "Подана" (модификация атрибута)
- Экземпляр app добавлен в коллекцию Applications сервиса GrantSystemService (формирование ассоциации)
- Выполнено логирование действия подачи заявки
- Экземпляр app возвращен как результат операции
