```plantuml
@startuml
actor "Независимый эксперт" as Expert
participant System
participant ApplicationList
participant Evaluations

Expert -> System : getAssignedApplications()
System -> ApplicationList : getAppList()
ApplicationList --> System : listOfApplications
System --> Expert : listOfApplications

loop для каждой заявки
    Expert -> System : openApplication(appID)
    System -> ApplicationList : loadDetails(appID)
    ApplicationList --> System : appDetails
    System --> Expert : appDetails

    Expert -> System : enterEvaluation(criteria, score, comments)
    System -> Evaluations : create(criteria, score, comments)
    Evaluations --> System : savedEvaluation
    System --> Expert : confirmation
end
ApplicationList --> System : updatedApp
System --> Expert : submissionOK
@enduml```
